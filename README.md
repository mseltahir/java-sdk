# Noebs Java Client        

[![](https://jitpack.io/v/tutipay/java-sdk.svg)](https://jitpack.io/#tutipay/java-sdk)

### Add it in your root build.gradle

  ```sh
  repositories {
			...
			maven { url 'https://jitpack.io' }
		}
  ```


  ### Add the dependency
  
   ```sh
dependencies {
	        implementation 'sd.noebs:java-sdk:LATEST_VERSION' # check latest version
	}
 ```
